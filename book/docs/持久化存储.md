# 持久化存储

`chrome.storage` 是用于数据持久化存储的核心 API ，相比传统的本地存储有更优的性能、更大的存储容量，且支持异步操作和分域存储。更新 `manifest.json` 文件内容如下。

```json
{
    "permissions": ["storage"]
}
```

第一种持久化存储方式是 `chrome.storage.local` ，数据仅保存在当前设备，无同步功能，数据永久保留，直到手动删除或扩展被卸载。更新 `content.js` 文件内容如下。

```javascript
const ChromeStorage = {
    set: data => {
        return new Promise((resolve, reject) => {
            chrome.storage.local.set(data, () => chrome.runtime.lastError ? reject(chrome.runtime.lastError) : resolve());
        });
    },
    get: keys => {
        return new Promise((resolve, reject) => {
            chrome.storage.local.get(keys, data => chrome.runtime.lastError ? reject(chrome.runtime.lastError) : resolve(data));
        });
    },
    remove: keys => {
        return new Promise((resolve, reject) => {
            chrome.storage.local.remove(keys, () => chrome.runtime.lastError ? reject(chrome.runtime.lastError) : resolve());
        });
    },
    clear: () => {
        return new Promise((resolve, reject) => {
            chrome.storage.local.clear(() => chrome.runtime.lastError ? reject(chrome.runtime.lastError) : resolve());
        });
    }
};

async function func() {
    try {
        let data = {
            name: 'Tom',
            age: 20
        };
        await ChromeStorage.set(data);
        console.log(await ChromeStorage.get(['name', 'age']));
        console.log(await ChromeStorage.get('name'));
        console.log(await ChromeStorage.get('age'));
        await ChromeStorage.remove('name');
        console.log(await ChromeStorage.get(['name', 'age']));
        await ChromeStorage.clear();
        console.log(await ChromeStorage.get(['name', 'age']));
    } catch (error) {
        console.error(error);
    }
}
func();
```

另一种是 `chrome.storage.sync` ，数据会通过用户的 Google 账号同步到其他设备，容量较小。
