# 匹配规则

匹配规则能精准控制扩展程序的生效范围，避免不必要的资源消耗和权限滥用。匹配规则的通用格式为：

```
<scheme>://<host><path>
```

`matches` 和 `exclude_matches` 是粗粒度匹配规则。更新 `manifest.json` 文件内容如下。

```json
{
    "content_scripts": [
	{
		"matches": ["<all_urls>"],
		"exclude_matches": ["https://*.github.io/*"]
    }
    ] 
}
```

`include_globs` 和 `exclude_globs` 是细粒度匹配规则。更新 `manifest.json` 文件内容如下。

```json
{
    "content_scripts": [
	{
        "matches": ["<all_urls>"],
		"include_globs": ["*github.com/search*q=chrome*"],
        "exclude_globs": ["*github.com/search*q=a*"]
    }
    ] 
}
```
