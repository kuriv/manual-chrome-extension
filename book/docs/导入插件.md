# 导入插件

在插件目录中新建 `popup.html` 文件内容如下。

```html
<!DOCTYPE html>
<html lang="zh-Hans">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Demo</title>
    <link rel="stylesheet" type="text/css" href="popup.css">
</head>
<body>
    <button id="btn">Click</button>
    <script type="text/javascript" src="popup.js"></script>
</body>
</html>
```

在插件目录中新建 `popup.css` 文件内容如下。

```css
#btn {
    background-color: #2563eb;
    border: none;
    border-radius: 5px;
    color: #fff;
    cursor: pointer;
    font-size: 14px;
    font-weight: 500;
    outline: 0;
    padding: 10px 20px;
    transition: all .2s ease;
    -webkit-tap-highlight-color: transparent;
}

#btn:hover {
    background-color: #1d4ed8;
    box-shadow: 0 2px 8px rgba(37, 99, 235, .2);
}

#btn:active {
    background-color: #1e40af;
    box-shadow: 0 1px 4px rgba(37, 99, 235, .15);
}
```

在插件目录中新建 `popup.js` 文件内容如下。

```javascript
const btn = document.getElementById('btn');
btn.addEventListener('click', () => alert('Hello World!'));
```

随后根据需要分别新建 `background.js` 、 `content.js` 、 `content.css` 文件。完成上述步骤后，此时打开 Chrome 浏览器管理扩展程序，勾选开发者模式，点击加载未打包的扩展程序按钮，选择刚才创建好的插件目录，即可查看效果。
