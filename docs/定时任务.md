# 定时任务

为避免 service worker 空闲时自动休眠，可以使用 `chrome.alarms` 。更新 `manifest.json` 文件内容如下。

```json
{
    "permissions": ["alarms"]
}
```

更新 `service-worker.js` 文件内容如下。

```javascript
chrome.runtime.onInstalled.addListener(() => {
    chrome.alarms.create('alarms', {
        delayInMinutes: 1,
        periodInMinutes: 1
    });
});

chrome.alarms.onAlarm.addListener(alarm => {
    if (alarm.name == 'alarms') {
        console.log(new Date().toLocaleString());
    }
});
```
