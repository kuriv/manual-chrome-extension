# 本地存储

content 注入到网页上下文中后，可以突破隔离访问原网页的本地存储。更新 `content.js` 文件内容如下。

```javascript
console.log(localStorage.getItem('foo'));
localStorage.setItem('foo', 'bar');
console.log(localStorage.getItem('foo'));
console.log(sessionStorage.getItem('baz'));
sessionStorage.setItem('baz', 'qux');
console.log(sessionStorage.getItem('baz'));
```
